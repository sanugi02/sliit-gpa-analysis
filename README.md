# SLIIT GPA Analysis System

A responsive web application to upload result sheet PDFs, calculate GPA,
rank students, and analyze academic performance.

## Features
- Upload multiple PDF result sheets
- Automatic GPA extraction & calculation
- Student ranking system
- Top 10 / 20 / 40 / 50 analysis
- Search student by ID
- Mobile & desktop responsive UI

## Tech Stack
- HTML, CSS, JavaScript
- PDF.js
- Git & GitHub
